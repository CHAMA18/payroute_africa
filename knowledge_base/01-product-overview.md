# PayRoute Africa Product Overview

## Summary

PayRoute Africa is a Flutter-based web application for cross-border payments, treasury visibility, wallet management, FX routing, onboarding, analytics, and operations across African markets.

The product presents itself as a modern fintech platform focused on:

- cross-border payments
- smart multi-rail routing
- cost optimization
- wallet and treasury management
- institutional onboarding
- payment operations dashboards

## Core Positioning

The landing experience emphasizes the following themes:

- borderless payments across Africa
- intelligent routing and resilience
- cost savings through route optimization
- enterprise-grade infrastructure
- fast, modern operational workflows

## Account Types

The application supports two main account types:

- `Personal Account`
- `Organization`

Personal accounts are framed around cross-border transfers and asset management.

Organization accounts are framed around:

- API-first architecture
- treasury controls
- bulk payouts
- cross-border infrastructure

## Primary User Areas

After authentication, the product exposes a dashboard-style application with these major areas:

- Dashboard
- Wallet
- Activity
- Cross-Border
- Smart Send
- Saving Goals
- ROI Analytics
- Exchange
- Cards
- Payment Links
- Settings

## Current Platform Stack

- Flutter web application
- Firebase Authentication
- Cloud Firestore
- Provider for state management
- GoRouter for navigation
- Docker-based deployment support
